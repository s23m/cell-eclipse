Using the containment tree viewer
---------------------------------

1. Import projects into Eclipse

2. Right click the org.s23m.cell.eclipse.visualization.containmenttree project, and select Run As > Eclipse Application

3. Inside the runtime workspace, select Window > Show View > Other... and choose S23M > Containment Tree.

4. Click the retrieve button

5. After the containment tree is loaded ("root" node appears), double-click an item to load it in the viewer.
